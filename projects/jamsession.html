Jam Session

Interactive Musician Forum

Hover for video with sound!

Jam Session is a web application for finding musicians and Artists via social media-like functionality. The goal is to create an app that makes finding musicians, groups, and gigs convenient while providing some interactivity and community.


From the user perspective, this app provides musicians a way to connect with each other. All skill levels and goals are represented, from beginner to expert and casual commitment to professional aspirations. Users can create a profile detailing their instruments played and relative skill level on each. They can display the genres of music they are interested in, level of commitment, and zip code in which they are located. Users can upload audio samples of their music to be displayed on their profile. Once a user has their profile set up, they can create posts advertising for themselves. They can also comment on posts and view the profiles of other users.


This web application was built with one other person. He focused on the backend and I focused on the frontend. I designed and built the prototype wireframes with Figma. I built the application using React with TypeScript, React Router, and Material UI. Some of the notable frontend features include: a customizable profile page, music and picture uploads for profile display, search functionality with filters.
