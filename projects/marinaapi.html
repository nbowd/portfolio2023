Marina API

Cloud API

Hover for video!

This is a cloud-based API to service a marina. Users can register new boats after they authorize their accounts through Google. Users can also create representations of loads that that boats will carry. Each load can only go on one boat but a boat will be able to carry multiple loads. The API supports full CRUD operations for both the boat and load entities, as well as endpoints to update the relationships between a load and a boat.


This API was built with Node.js and hosted on Google Cloud Platform through Google App Engine. It was containerized using Docker. Data is stored using Google Cloud Datastore for a scalable solution. OAuth was provided and handled by the Google People API through the use of JWTs in requests. Development of this project was primarily done with Test Driven Development(TDD) using Postman tests. Thoroughly documented in API spec.
